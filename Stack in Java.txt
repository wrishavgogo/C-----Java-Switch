Declaration : 

Stack<Type> stack = new Stack<>(); 


Key Methods :- 
1. push(E item): Adds an item to the top of the stack.
2. pop(): Removes and returns the item from the top of the stack.
3. peek(): Returns the item at the top of the stack without removing it.
4. isEmpty(): Checks if the stack is empty.
5. search(Object o): Returns the 1-based position of the object in the stack. Returns -1 if the object is not found. ---> This is new functionality of search , Not present in C++
6. size(): Returns the number of elements in the stack.
